# ScreenLeak-Detection
Pixel-wise detection, using color space transformation to Y space.
Used Ymax, uniformity, and Ylocals to find screen leakage.

# C++ Version
- 3rdParty: 
- Boost: 1.68
- OpenCV: 3.4.3
